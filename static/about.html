THE PROJECT

AI diagnostics for
every farmer

CropSense is a deep learning project that puts plant disease detection in the hands of anyone with a smartphone camera. Trained on tens of thousands of leaf images, it identifies 23 disease conditions across 8 crop families with near-expert accuracy — instantly and for free.

OUR MISSION
"Making accurate crop disease diagnosis accessible to every farmer, everywhere — powered by open-source AI."
WHY CROPSENSE

The problem we're solving

Crop diseases destroy an estimated 20–40% of global agricultural production each year. Early and accurate diagnosis is the most impactful intervention.

⚡

Instant Diagnosis

Traditional disease identification requires physical lab samples and days of waiting. CropSense delivers results in under one second from any photo.

🎯

High Accuracy

Our fine-tuned ResNet-18 model achieves ~96% validation accuracy on the PlantVillage benchmark — comparable to expert agronomist performance.

🌍

Accessible

No special equipment needed. Any photo taken with a standard smartphone camera in good natural lighting is sufficient for reliable diagnosis.

🔓

Open Source

The full training pipeline, model weights, and web application are available on GitHub under an open license for the research community.

🧠

Transfer Learning

By building on ImageNet pre-trained weights, the model achieves excellent accuracy even with the limited scope of the PlantVillage dataset.

📱

Web-Native

No app installation required. The diagnostic tool runs entirely in your browser via FastAPI and can be hosted on any server or local machine.

DEVELOPMENT JOURNEY

How the project was built

From dataset research to a fully functional web application.

PHASE 01

Dataset Preparation

Downloaded and organized the PlantVillage dataset into 23 category folders. Implemented a custom dataset.py using ImageFolder with a 70/15/15 train/val/test split and reproducible seeding.

PHASE 02

Model Design

Chose ResNet-18 for its ideal balance between accuracy and inference speed. Replaced the final fully-connected layer with a 23-class linear layer and loaded ImageNet pre-trained weights.

PHASE 03

Training & Evaluation

Fine-tuned using Adam optimizer with CrossEntropyLoss. Saved the best model checkpoint based on validation loss. Achieved ~96% validation accuracy on the held-out test set.

PHASE 04

FastAPI Integration

Built a /predict POST endpoint that accepts image uploads, runs the full preprocessing pipeline and inference in-memory, and returns a JSON response with class name and confidence.

PHASE 05

Frontend & UI

Designed a premium dark-theme web interface with drag-and-drop upload, animated confidence bars, a tab component for diagnosis details, and a full results view — all in vanilla HTML/CSS/JS.
